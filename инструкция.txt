создать окружение Python:
python -m venv .venv

запустить виртуальное окружение:
.venv/Scripts/activate

установить зависимости:
pip install -r requirements.txt

запустить сервер:
для хостинга: uvicorn main:app --host 0.0.0.0 --port 8000
локальное тестирование: uvicorn main:app --reload

установить localtunnel (требуется node.js):
npm install -g localtunnel

запустить localtunnel (в отдельном терминале):
lt --port 8000

получить localtunnel пароль:
curl https://loca.lt/mytunnelpassword
