# Спецификация: Визуальные активы (спрайты)

## Цель
Определить требования к графическим ресурсам для первой итерации и дальнейшего развития.

## Подход к первой итерации (MVP)
- Минимализм: все объекты рисуются кодом через Canvas (векторные примитивы).
- Спрайты (растровые изображения) не используются — только геометрические фигуры и цвета.
- Это позволяет быстро начать разработку, не дожидаясь художника.

## Текущие графические примитивы

### Игроки (по фракциям)
| Фракция      | Форма          | Цвет (hex) |
|--------------|----------------|------------|
| Фидерум      | Ромб           | #3366FF    |
| Дин'Аслан    | Полукруг       | #FFCC00    |
| Фриктоз      | Круг с шипами  | #CC3333    |
| Апиро-Кипера | Капля          | #9933FF    |
| Адапториум   | Овал           | #33CC66    |
| Последний Патч| Шестиугольник  | #999999    |
| С.Л.О.Р.     | Квадрат        | #666666    |
| Церковь      | Треугольник    | #CC6600    |

- Шипы, глаза и детали пока не рисуем — только базовый контур.
- Размер: вписаны в окружность радиусом 25.

### Оружие
- Серая линия (длина 30, толщина 5) от центра игрока по направлению угла.
- На конце маленький кружок (прицел) — опционально.

### Пули
- Серый круг радиусом 5.

### Мобы
- Электрофаг: ярко-жёлтый круг радиусом 20 с пульсацией (изменение яркости).

### Стены
- Тёмно-серый прямоугольник с чёрной обводкой.

### Интерфейс
- Шрифт моноширинный (например, Share Tech Mono Cyrillic).
- Цвет текста: зелёный/оранжевый на чёрном фоне.

## Вторая итерация (когда появится художник)
- Спрайты в формате PNG с прозрачностью.
- Для каждой фракции — уникальный дизайн с глазами и деталями.
- Анимации (движение, выстрелы, смерти).
- Эффекты (вспышки, дым).

## Требования к спрайтам (будущим)
- Размер: ~64x64 пикселя для игроков.
- Поддержка поворота (спрайт рисуется под любым углом).
- Единый стиль — плоский, минималистичный, с цветовой кодировкой фракций.

## Утилиты
- Для прототипирования можно использовать Figma для создания простых векторных форм и экспорта в PNG.
