# Roadmap MVP: от нуля до играбельного прототипа
## Версия 0.1 — "Скелет"
### Цель: Убедиться, что сеть работает, игроки подключаются, видят друг друга и могут двигаться.

- **Сеть:** WebSocket сервер на Python (FastAPI), клиент подключается.
- **Комнаты:** Одна комната на 4 игрока, автоматический старт при заполнении.
- **Модель игрока:** ID, координаты, угол поворота, модели (базовые формы).
- **Движение:** WASD + мышь (угол), сервер обновляет позиции и рассылает всем.
- **Отрисовка:** Canvas, цветные кружки/формы для игроков, камера следует за своим игроком.
- **Карта:** Просто белое сетчатое поле (как тетрадный лист) для отслеживания движения, с границами (нельзя выйти за края).
- **Результат:** Четыре игрока могут бегать по полю и видеть друг друга.
