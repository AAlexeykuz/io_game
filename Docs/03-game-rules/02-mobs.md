# Спецификация: Мобы (нейтральные существа)

## Цель
Добавить на карту простых мобов для фарма очков и разнообразия геймплея.

## Тип моба (первая итерация)
- Только один тип: **Электрофаг**.
- Внешний вид: маленький светящийся шарик (позже спрайт).
- Поведение: случайное блуждание по карте.

## Параметры
- `MOB_SPEED = 2` единицы за тик.
- `MOB_RADIUS = 20` (для коллизий).
- `MOB_HEALTH = 1` (умирает с одного попадания).
- `MOB_SCORE = 1` очко за убийство (см. scoring.md).
- Максимальное количество мобов на карте: `MAX_MOBS = 10`.

## Спавн мобов
- Мобы появляются случайно на карте, но не в стенах и не слишком близко к игрокам (радиус `MIN_SPAWN_DISTANCE_FROM_PLAYERS = 200`).
- Сервер поддерживает количество мобов: если число мобов упало ниже `MAX_MOBS`, через случайный интервал (5–10 секунд) спавнится новый моб.

## Движение
- На каждом тике моб меняет направление с вероятностью `CHANGE_DIR_PROB = 0.05`.
- Направление задаётся углом (0–360), скорость постоянна.
- При столкновении со стеной моб отталкивается (меняет направление на противоположное или случайное).

## Взаимодействие с игроками
- Мобы не атакуют игроков (пока пассивны).
- При попадании пули моб теряет здоровье и умирает (исчезает).

## Выходные данные
- Список мобов передаётся клиентам в поле `mobs` состояния (см. `02-networking/01-protocol.md`).
